Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Вывод: сначала программа произведет печать значений, после чего упадет с ошибкой.

Ключевое слово range работает с каналами, однако цикл range завершается только тогда, когда канал закрыт или использовано ключевое слово break. В данном случае break'а нет - как следствие получаем бесконечный цикл с ожидаем значения из канала.

```
