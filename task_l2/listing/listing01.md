Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Вывод: [77,78,79]

Создастся слайс от исходного массива ([77,78,79]). Следует отметить, что оба объекта ссылаются на одну и ту же область памяти - как следствие при изменении одного объекта, будет менятся другой (до тех пор, пока не произойдет перевыделение памяти под элементы - тогда они начнут ссылаться на разные области памяти).
```
